Any Seismic Data Format! Anytime! Anywhere! 

>>> from obspy import read
                                            >>> st = read("http://examples.obspy.org/IU_ULN_2015.mseed")
                                            >>> st.plot()
  • Can read (and mostly write) (Mini)SEED, SAC, SEG-Y, AH (Ad Hoc), ASCII, CSS, DataMark (Win), GSE2/GSE1, Kinemetrics Evt, K-NET and KiK-net, PDAS, SEISAN, SEG-2, SU (Seismic Unix), Q and ASC (Seismic Handler), WAV, and Nanometrics Y files.
  • Automatic Format Detection!
  • Generally Awesome!

Access Data From All Over the World 

>>> from obspy.clients.fdsn import Client
                                            >>> c = Client("IRIS")
                                            >>> c.get_stations(network="IU").plot();
  • Access waveform, station, and event data.
  • Supports the FDSN web services (IRIS, ORFEUS, Geonet, RESIF, INGV, ETH, GFZ, ...), ArcLink, Filesystems (e.g. SDS), NEIC, SeisHub, Earthworm, SeedLink, and the IRIS Syngine service.
  • Fully integrated with the rest of ObsPy!
  • Large scale downloading with the mass downloader.

Comprehensive Signal Processing Toolbox 

>>> from obspy import read, read_inventory
                                            >>> st = read("http://examples.obspy.org/IU_ULN_2015-07-18T02.mseed")
                                            >>> inv = read_inventory("http://examples.obspy.org/IU_ULN.xml")
                                            >>> st.remove_response(inv, output="VEL")
  • Instrument correction from StationXML, SEED, RESP, and custom data structures.
  • Signal filtering, tapering, detrending, resampling, and much more.
  • Advanced algorithms, e.g. array analysis, multi-station triggers, ...
  • Direct access to the full power of the scientific Python ecosystem.
